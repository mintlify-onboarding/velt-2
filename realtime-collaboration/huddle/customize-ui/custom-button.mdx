## Custom Huddle Button

If you want to replace the default arrow button with your own custom button, you can pass it in as a child component.

```jsx
<VeltHuddleTool>
    <button className="myButton"></button
</VeltHuddleTool>
```

## Custom Huddle Button

If you want to replace the default arrow button with your own custom button, you can pass it in as a child component.

```html HTML
<velt-arrow-tool>
    <button class="myButton"></button
</velt-arrow-tool>
```
