## Replace the Sidebar Button
**Provide a template for the sidebar button.**

Target the button slot with your own custom template.


```jsx
<VeltSidebarButton>
  <div slot="button">
      <!-- YOUR Custom HTML here -->
  </div>
</VeltSidebarButton>
```



<RequestExample>

```js React / Next.js
import {
  VeltSidebarButton
} from '@veltdev/react';

export default function App() {

  return (
    <>

      <VeltSidebarButton>
        <div slot="button">
            <!-- YOUR Custom HTML here -->
        </div>
      </VeltSidebarButton>

    </>
  );
}
```
