## `TagAnnotation` Class Properties

| Property           | Type                    | Optional | Description                                                      |
|--------------------|-------------------------|----------|------------------------------------------------------------------|
| `annotationId`       | String                  | No       | A unique identifier for the tag annotation, auto-generated.      |
| `from`               | `User`                    | No       | The user who created the tag annotation.                         |
| `to`                 | `User[]`                  | No       | List of users that are tagged in the annotation.                 |
| `color`              | String                  | Yes      | The color used for the tag annotation.                           |
| `resolved`           | Boolean                 | Yes      | Indicates whether the tag annotation is marked as resolved.      |
| `lastUpdated`        | Any                     | Yes      | Timestamp of the last update to the annotation, auto-generated.  |
| `positionX`          | Number                  | Yes      | X-axis position of the annotation, auto-generated.               |
| `positionY`          | Number                  | Yes      | Y-axis position of the annotation, auto-generated.               |
| `screenWidth`        | Number                  | Yes      | The user’s screen width, auto-generated.                         |
| `screenHeight`       | Number                  | Yes      | The user’s screen height, auto-generated.                        |
| `screenScrollHeight` | Number                  | Yes      | The user’s screen scroll height, auto-generated.                 |
| `screenScrollTop`    | Number                  | Yes      | The user’s screen scroll top, auto-generated.                    |
| `taggedElementPath`  | String                  | Yes      | The XPath of the clicked element for tagging, auto-generated.    |
| `taggedElementRect`  | Any                     | Yes      | The rect of the clicked element for tagging, auto-generated.     |
| `targetElement`      | TargetElement \| Null   | Yes      | The target element of the attached annotation.                   |
| `position`           | CursorPosition \| Null  | Yes      | The cursor position related to the annotation.                   |
| `locationId`         | Number \| Null          | Yes      | Unique ID for the location generated from the provided location. |
| `location`           | `Location` \| Null        | Yes      | The location used to identify a user on a sub-document.          |
| `type`               | String                  | Yes      | The type of annotation.                                          |
| `targetTextRange`    | TargetTextRange \| Null | Yes      | The range of text that the annotation is related to.             |
| `approved`           | Boolean                 | Yes      | The approved status of the annotation.                           |
| `annotationIndex`    | Number                  | Yes      | Index of the annotation in the list of annotations.              |
| `pageInfo`           | `PageInfo`                | Yes      | Information about the page where the annotation is made.         |
