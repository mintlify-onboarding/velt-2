## `TargetElement` Class Properties

| Property        | Type    | Optional | Description                                                        |
|-----------------|---------|----------|--------------------------------------------------------------------|
| `xpath`           | String  | Yes      | The XPath of the target element.                                   |
| `fXpath`          | String  | Yes      | The full XPath of the target element.                              |
| `topPercentage`   | Number  | No       | The relative top position of the cursor on the target element.     |
| `leftPercentage`  | Number  | No       | The relative left position of the cursor on the target element.    |
