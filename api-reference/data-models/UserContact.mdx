## `UserContact` Class Properties

| Property    | Type     | Optional | Description                                                                                                   |
|-------------|----------|----------|---------------------------------------------------------------------------------------------------------------|
| `userId`      | String   | No       | Unique identifier used to identify the user.                                                                  |
| `name`        | String   | Yes      | The user's full name. Defaults to a random avatar name if not provided.                                       |
| `photoUrl`    | String   | Yes      | URL for the user's display picture. Defaults to a random avatar image if not provided.                        |
| `email`       | String   | No       | Required for sending email or Slack notifications related to comments and tags.                               |
| `groupId`     | String   | Yes      | An identifier used to cluster groups of users, such as a domain name.                                         |
| `source`      | String   | Yes      | The origin of the contact, or the method by which the contact was added or shared.                            |
| `visibility`  | String   | No       | Specifies the visibility of the contact; can be 'group' for group visibility or 'private' for individual use. |
