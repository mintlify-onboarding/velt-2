## `PageInfo` Class Properties

| Property      | Type          | Optional | Description                                       |
|---------------|---------------|----------|---------------------------------------------------|
| `url`           | String        | Yes      | The full URL of the webpage                       |
| `path`          | String        | Yes      | The path of the webpage, excluding the base URL   |
| `baseUrl`       | String        | Yes      | The base URL (domain) of the webpage              |
| `title`         | String        | Yes      | The title of the webpage                          |
| `arrowUrl`      | String        | Yes      | The reference URL for an arrow annotation         |
| `areaUrl`      | String        | Yes      | The reference URL for an area annotation          |
| `commentUrl`    | String        | Yes      | The reference URL for a comment annotation        |
| `tagUrl`        | String        | Yes      | The reference URL for a tag annotation            |
| `recorderUrl`   | String        | Yes      | The reference URL for a recorder annotation       |
| `screenWidth`   | Number        | Yes      | The width of the user's screen (auto-generated)   |
| `deviceInfo`    | `IDeviceInfo`   | Yes      | Information about the user's device               |
