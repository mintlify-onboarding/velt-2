

## `Transcription` Class Properties

| Property             | Type    | Optional | Description                                               |
|----------------------|---------|----------|-----------------------------------------------------------|
| `from`                 | User    | No       | The user who created the transcription.                   |
| `lastUpdated`          | Number  | Yes      | Timestamp of when the transcription was last updated.     |
| `srtBucketPath`        | String  | Yes      | The cloud storage bucket path for the SRT file.           |
| `srtUrl`               | String  | Yes      | The URL to access the SRT file.                           |
| `transcriptedText`     | String  | Yes      | The text that has been transcribed.                       |
| `transcriptionLatency` | Number  | Yes      | The time taken to transcribe the recording, in milliseconds. |
| `vttBucketPath`        | String  | Yes      | The cloud storage bucket path for the VTT file.           |
| `vttUrl`               | String  | Yes      | The URL to access the VTT file.                           |
