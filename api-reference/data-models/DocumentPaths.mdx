## `DocumentPaths` Class Properties

| Property                           | Type    | Optional | Description                                                  |
|------------------------------------|---------|----------|--------------------------------------------------------------|
| `documentId`                       | string  | Yes      | The unique identifier for the document.                      |
| `presence`                         | string  | Yes      | Path for user presence information.                          |
| `cursor`                           | string  | Yes      | Path for user cursor position information.                   |
| `area`                             | string  | Yes      | Path for defining an area within the document.               |
| `arrow`                            | string  | Yes      | Path for arrow annotations or pointers.                      |
| `comment`                          | string  | Yes      | Path for comments added to the document.                     |
| `audioHuddle`                      | string  | Yes      | Path for audio-based collaboration or 'huddle' sessions.     |
| `clientDocumentId`                 | string  | Yes      | Document identifier provided by the client.                  |
| `groupContacts`                    | string  | Yes      | Path for contact information for groups.                     |
| `globalContacts`                   | string  | Yes      | Path for global contact information.                         |
| `logins`                           | string  | Yes      | Path for user login information.                             |
| `selection`                        | string  | Yes      | Path for user selection within the document.                 |
| `tag`                              | string  | Yes      | Path for tags associated with parts of the document.         |
| `huddle`                           | string  | Yes      | Path for collaboration or 'huddle' sessions.                 |
| `metadata`                         | string  | Yes      | Path for document metadata.                                  |
| `recorder`                         | string  | Yes      | Path for recording document sessions or changes.             |
| `iam`                              | string  | Yes      | Path for Identity and Access Management configurations.      |
| `documentIam`                      | string  | Yes      | Path for document-specific IAM settings.                     |
| `users`                            | string  | Yes      | Path for document user information.                          |
| `userFeedback`                     | string  | Yes      | Path for user feedback submissions.                          |
| `userReportBugs`                   | string  | Yes      | Path for user bug report submissions.                        |
| `userContactUs`                    | string  | Yes      | Path for users to contact the service provider.              |
| `flock`                            | string  | Yes      | Path for user 'follow along' sessions.                       |
| `syncVideoPlayer`                  | string  | Yes      | Path for synchronizing a video player among users.           |
| `liveState`                        | string  | Yes      | Path for the live state of the document.                     |
| `liveStateSingleEditorMode`        | string  | Yes      | Path for the live state when in single editor mode.          |
| `reaction`                         | string  | Yes      | Path for user reactions within the document.                 |
