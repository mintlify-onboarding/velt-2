## `Selection` Class Properties

| Property        | Type                   | Optional | Description                                                   |
|-----------------|------------------------|----------|---------------------------------------------------------------|
| `user`            | `User`                   | Yes      | User focused on specific element or text.                     |
| `type`            | 'focus' \| 'click' \| 'selection' | Yes      | Type of event through which the highlight event is fired.     |
| `targetElement`   | String \| null         | Yes      | XPath of the highlighted element.                             |
| `targetTextRange` | TargetTextRange \| null | Yes      | Selected text to highlight.                                   |
| `pageInfo`        | `PageInfo`               | No       | Information about the page where the selection is made.       |
