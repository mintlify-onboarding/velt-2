## `ScreenSize` Class Properties

| Property           | Type    | Optional | Description                           |
|--------------------|---------|----------|---------------------------------------|
| `screenWidth`        | Number  | No       | User's current screen width.          |
| `screenHeight`       | Number  | No       | User's current screen height.         |
| `screenScrollHeight` | Number  | No       | User's current screen scroll height.  |
