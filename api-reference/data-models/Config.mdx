## `Config` Class Properties

| Property                  | Type               | Optional | Default                                               | Description                                                                                                                            |
|---------------------------|--------------------|----------|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------|
| `urlAllowList`            | string[]           | Yes      | All pages                                             | Restricts Snippyly features to specific pages by specifying partial URL strings.                                                       |
| `featureAllowList`        | FeatureType[]      | Yes      | All features                                          | Only allows the provided Snippyly features to run.                                                                                     |
| `userPlanAllowList`       | string[]           | Yes      | All users                                             | Restricts Snippyly features to specific user plans.                                                                                    |
| `userIdAllowList`         | string[]           | Yes      | All users                                             | Restricts Snippyly features to specific users.                                                                                         |
| `userGroupIdAllowList`    | string[]           | Yes      | All groups/teams                                      | Restricts Snippyly features to specific groups/teams of users.                                                                         |
| `debugMode`               | boolean            | Yes      | -                                                     | Enables or disables debug mode. This change will be reflected after a page refresh.                                                    |
| `usePrefersColorScheme`   | boolean            | Yes      | -                                                     | If set to true, listens to changes on the `prefers-color-scheme` media query to set the global theme of Snippyly components.           |
