



## `DocumentMetadata` Properties

| Property            | Type                        | Optional | Description                                      |
|---------------------|-----------------------------|----------|--------------------------------------------------|
| `documentId`        | `string`                    | Yes      | Unique document id generated from client document id |
| `clientDocumentId`  | `string` \| `number`        | Yes      | Document id provided by client                   |
| `pageInfo`          | `PageInfo`                  | Yes      | Page metadata                                    |
| `locations`         | `{ [locationId: number]: LocationMetadata; }` | Yes | Object containing location metadata keyed by location ID |
