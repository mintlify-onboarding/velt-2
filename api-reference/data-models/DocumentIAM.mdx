## `DocumentIAM` Class Properties

| Property              | Type                            | Optional | Description                                           |
|-----------------------|---------------------------------|----------|-------------------------------------------------------|
| `documentAccessType`  | `DocumentAccessType`            | No       | The type of access allowed for the document.          |
| `features`            | `DocumentIAMFeatures \| undefined` | Yes      | Features relating to document access, may be undefined.|
| `users`               | `{ [userId: string]: UserIAM}`  | No       | Object containing `UserIAM` objects with the user's ID string as the key. |

## `DocumentIAMFeatures` Type Properties

| Property | Type    | Optional | Description                                           |
|----------|---------|----------|-------------------------------------------------------|
| `all`    | Object  | Yes      | Object containing features applicable to all users.   |

| Property       | Type    | Optional | Description                                      |
|----------------|---------|----------|--------------------------------------------------|
