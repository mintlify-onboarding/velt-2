


<Steps>
<Step title="getAllTagAnnotations()">
This method retrieves all tag annotations.
```jsx
const tagElement = client.getTagElement();
tagElement.getAllTagAnnotations();
```
</Step>
<Step title="deleteTag()">
This method deletes a tag.
```jsx
const tagElement = client.getTagElement();
tagElement.deleteTag();
```
</Step>
<Step title="allowedElementIds()">
This method retrieves allowed element IDs.
```jsx
const tagElement = client.getTagElement();
tagElement.allowedElementIds();
```
</Step>
<Step title="attachTag()">
This method attaches a tag.
```jsx
const tagElement = client.getTagElement();
tagElement.attachTag();
```
</Step>
<Step title="addTagModeChange()">
This method adds a tag mode change.
```jsx
const tagElement = client.getTagElement();
tagElement.addTagModeChange();
```
</Step>
